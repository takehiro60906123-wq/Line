## 2. ステータス体系（5種）

| ステータス | 略称 | 役割 |
|-----------|------|------|
| HP | HP | 体力。0で戦闘不能 |
| ATK | ATK | 攻撃力。通常攻撃・スキル両方で使用 |
| DEF | DEF | 防御力。通常攻撃のダメージを軽減 |
| SPD | SPD | 素早さ。行動順を決定 |
| RES | RES | 魔防。スキル攻撃のダメージを軽減 |

### HP帯のスケール
- **HPは他ステータスの約5〜6倍のスケール**
  - HP: 150〜600帯
  - ATK/DEF/SPD/RES: 20〜170帯
- これにより「数ターンの殴り合い」が成立する

### 設計意図
- ATKは1本化。アタッカーはATKだけ見ればいいのでシンプル
- 壁キャラだけがDEF型（通常攻撃に強い）とRES型（スキルに強い）に分かれる
- ユーザーは「ATK高い＝強い攻撃キャラ」と直感的に理解できる

## 3. ダメージ計算式

### 通常攻撃（毎ターン必ず発生）
```
ダメージ = (ATK / DEF) × 50 × 乱数(0.85〜1.0) × 属性相性倍率
```

### スキル攻撃（チャージ完了時に発動）
```
ダメージ = (ATK × スキル威力% / RES) × 50 × 乱数(0.85〜1.0) × 属性相性倍率
```

### 計算例（×50の係数での期待値）

| 場面 | ATK | DEF/RES | 倍率 | ダメージ |
|------|-----|---------|------|---------|
| 見習い剣士→見習い盾兵 | 80 | 60 | 1.0 | 67 |
| 英雄剣士→見習い盾兵 | 150 | 60 | 1.0 | 125 |
| 英雄剣士→英雄盾兵 | 150 | 120 | 1.0 | 63 |
| 英雄剣士→英雄盾兵（必殺技） | 150 | 120 | 1.8 | 113 |
| 見習い→伝説王 | 80 | 150 | 1.0 | 27 |

> **注**: ×50の係数はプロトタイプテストで調整する。HP帯300〜600で数ターンかかるバランスが理想。

## 5. シェイプ別行動回数（DQM参考）

### 基本ルール
| シェイプ | 枠数 | 行動回数 | 対応人材 |
|---------|------|---------|---------|
| 1x1 | 1枠 | **1回（固定）** | 一般兵、専門家 |
| 1x2 / 2x1 | 2枠 | **1〜2回（ランダム）** | 精鋭、英雄級 |
| 2x2 | 4枠 | **1〜3回（ランダム）** | 伝説の英雄、王 |

### 行動回数の確率テーブル

**2枠（1〜2回行動）**:
| 回数 | 確率 |
|------|------|
| 1回 | 40% |
| 2回 | 60% |
期待値: 1.6回

**4枠（1〜3回行動）**:
| 回数 | 確率 |
|------|------|
| 1回 | 30% |
| 2回 | 50% |
| 3回 | 20% |
期待値: 1.9回

### 重要制限: スキルは1ターン1回まで
```
行動回数が複数回の場合:
  1回目 → 通常攻撃 または スキル（チャージ完了時）
  2回目以降 → 通常攻撃のみ（ダメージ×0.7減衰）
```

### ダメージ減衰ルール（複数行動時）
| 行動 | ダメージ倍率 | 備考 |
|------|------------|------|
| 1回目 | ×1.0 | 通常 or スキル |
| 2回目 | ×0.7 | 通常攻撃のみ |
| 3回目 | ×0.7 | 通常攻撃のみ |

> **設計意図**: 複数枠キャラはBST自体が高いため、行動回数が等倍のままだと1枠4人を大きく上回る。2回目以降に×0.7の減衰を入れることで、複数枠の爆発力を抑えつつ「当たり回を引けば強い」ロマンを残す。

### バランス検証（減衰込み）
```
4枠1人: ATK150 × (1.0 + 0.9×0.7 + 0.2×0.7) = ATK150 × 1.77 = 実質266相当
1枠4人: ATK70 × 4.0回 = 280相当
```
1枠4人がやや上回る。4枠は「1人で4枠分の仕事をする」利便性がメリット。壁分散できない弱点とのトレードオフ成立。

## 6. スキルシステム

### 基本仕様
- **チャージ制**（現行システム継続）。Nターンでチャージ完了→発動→命中判定
- キャラ1人に**スキル1〜2個**
- 昇格で新スキルを覚える（追加 or 上位版に差し替え）
- スキルダメージは**ATK × 威力倍率 vs RES**で計算

### スキルのパラメータ
```javascript
{
  name: "烈火斬",
  type: "SINGLE",       // ターゲット種別
  element: "fire",      // 属性
  power: 80,            // 威力（%）
  accuracy: 90,         // 命中率（%）
  charge: 3,            // チャージターン数
  effect: null,         // 追加効果
  effectRate: 0,        // 追加効果発生率（%）
  desc: "炎を纏った斬撃"
}
```

## 7.5. リーダースキルシステム（新規）

### 概要
- 8枠の**1番目に配置したキャラ**がリーダーとなる
- リーダーは固有の「リーダースキル」を持ち、**部隊全体に効果**を発揮
- 個別アビリティ（豪腕、鉄壁等）とは**別枠で重複する**
- **勝敗を左右するレベルの強力な効果**。編成の最重要要素

### 基本ルール
- 全キャラがリーダースキルを持つ（一部は弱い汎用型）
- 条件が厳しいほど効果が強い
- 条件を満たさないメンバーにはリーダースキルが適用されない
- リーダーが戦闘不能になっても効果は継続（開戦時に適用済み）

### リーダースキルのデータ構造
```javascript
{
  leaderSkill: {
    name: "炎の覇道",
    desc: "火属性の味方全員HP+30%, ATK+40%",
    condition: { element: "fire" },     // 適用条件
    effects: [
      { stat: "hp",  mul: 1.30 },
      { stat: "atk", mul: 1.40 }
    ]
  }
}
```

### 条件タイプ
| conditionタイプ | 意味 | 例 |
|----------------|------|-----|
| `{ element: "fire" }` | 指定属性のみ | 火属性全員 |
| `{ gender: "F" }` | 指定性別のみ | 女性全員 |
| `{ all: true }` | 無条件で全員 | 全員に適用 |
| `{ elementCount: {element:"dark", min:3} }` | 指定属性N人以上で全員 | 闇3人以上で全員 |
| `{ genderCount: {gender:"F", min:4} }` | 指定性別N人以上で全員 | 女性4人以上で全員 |
| `{ multiElement: {min:3} }` | N種以上の属性で全員 | 3属性以上で全員 |
| `{ allElement: true }` | 6属性全部入りで全員 | 全属性揃い |
| `{ shape: "S1" }` | 指定シェイプのみ | 1x1のみ |
| `{ elements: ["fire","water"] }` | 複数属性指定 | 火と水のみ |

> **設計意図**: リーダースキルが勝敗を左右するレベルの強さなので、「誰をリーダーにするか」「条件を満たす編成を組めるか」が戦略の核になる。属性染めvs混成、男性パvs女性パなど、複数の最適解が存在する状態が理想。

### 実装影響範囲

| data.js | 各キャラに`leaderSkill`フィールド追加 |

### 調整つまみ（何をいじれば何が変わるか）
| 問題 | 調整箇所 |
|------|---------|
| 全体的にダメージが高すぎる/低すぎる | 計算式の係数（×50）を変更 |
| 4枠が強すぎる | 行動回数の確率テーブル or 減衰倍率(×0.7)を下げる |
| 4枠が弱すぎる | 減衰倍率を上げる(×0.8等) or BST上方修正 |
| 特定の特性が壊れてる | 特性の数値を下げる |
| スキルが強すぎる | チャージターンを伸ばす or 命中率を下げる |
| 壁が硬すぎる | DEF/RESの上限を設定 |
| 壁が意味ない | 計算式のDEF/RES影響度を上げる |
| 属性相性が強すぎる | 倍率を1.3/0.85等に緩和 |
| リーダースキルが強すぎる | 属性染め系の倍率を下げる(+40%→+30%等) |
| リーダースキルが弱すぎる | 汎用型の倍率を上げる / 条件を緩和 |
| 属性染めが最適解すぎる | 混成条件系の効果を上げる |
| 性別条件が不公平 | 150人中の性別比率を調整 |

### 重要ルール
- スキルは1ターンに1回まで（複数行動キャラ用の制限）
- 複数行動の2回目以降はダメージ×0.7減衰
- 状態異常は行動阻害系は相互排他（スタン/睡眠/麻痺/混乱は同時にかからない）
- 会心ダメージは×1.5固定
- 最低ダメージは1（0にはならない）
- リーダースキルは戦闘開始時に1回適用（リーダー戦闘不能後も継続）
- リーダースキルと個別アビリティは別枠で重複する
